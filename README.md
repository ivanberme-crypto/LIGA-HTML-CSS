# FUTDRAFT

FutDraft es un proyecto desarrollado para la asignatura de **Lenguaje de Marcas**. Consiste en una página web sobre fútbol que permite navegar entre diferentes ligas europeas, ver equipos, jugadores y distintas secciones relacionadas con el fútbol.

El proyecto está desarrollado utilizando **HTML, CSS y JavaScript** para crear una página web estructurada, interactiva y visualmente organizada.



# CARACTERÍSTICAS

- Página web desarrollada con **HTML**
- Diseño y estilos mediante **CSS**
- Interactividad mediante **JavaScript**
- Navegación entre distintas **ligas europeas**
- Visualización de **equipos y jugadores**
- Menú desplegable de navegación
- Carrusel de imágenes
- Organización clara del contenido

# PARA EJECUTARLO

1. Descargar el proyecto o clonar el repositorio.

2. Abrir la carpeta del proyecto.

3. Ejecutar el archivo:


Index.html


4. El navegador mostrará la página principal del proyecto.


# OBJETIVO DEL PROYECTO

El objetivo principal del proyecto es practicar el uso de **Lenguaje de Marcas**, aplicando:

- Creación de páginas web con **HTML**
- Diseño y estilos mediante **CSS**
- Interactividad utilizando **JavaScript**
- Organización estructurada de un proyecto web
- Navegación entre diferentes páginas
